# 🍕 Laravel Food Delivery System

A modern food delivery platform built with **Laravel 13**, **Vue 3**, and **Inertia.js**.

---

## ✨ Features

- 🔐 **Authentication** - Login, Register, Password Reset
- 👤 **User Roles** - Admin, Vendor, Staff, Customer
- 🏪 **Restaurant Management** - CRUD operations
- 📦 **Order Management** - Place and track orders
- 🛡️ **RBAC** - Role-based access control

---

## 🛠️ Tech Stack

| Backend | Frontend |
|---------|----------|
| Laravel 13 | Vue 3 |
| MySQL | Inertia.js |
| Laravel Breeze | Tailwind CSS |
| Laravel Sanctum | Vite |

---

## 📋 Prerequisites

- PHP >= 8.2
- Composer
- Node.js >= 18
- MySQL

---

## 🔧 Installation

```bash
# 1. Clone the repository
git clone https://github.com/zeeshanbhatti0109/Laravel-Food-Delivery-Project.git
cd Laravel-Food-Delivery-Project

# 2. Install PHP dependencies
composer install

# 3. Setup environment
cp .env.example .env
php artisan key:generate

# 4. Configure database in .env file

# 5. Install NPM dependencies
npm install

# 6. Build assets
npm run build

# 7. Run migrations & seeders
php artisan migrate --seed

# 8. Start the server
php artisan serve
npm run dev  # In a separate terminal
